Question:
Find the sum of ODD SQUARES :

## Formulas

* Sum till n → `n*(n+1)/2`  `[1,n]`
* Sum of squares → `n*(n+1)*(2*n+1)/6`
* Sum of odds till n → use AP
  `(1,3,5,7 → s(n) = n/2*(2+(n-1)*2) = n/2*(2*n) = n^2)`

---

## Setup

* `SUM = SM`
* odd no = `2*i + 1`

---

## Goal

Calculate:

```
SM = Σ (2*i + 1)^2   for i ∈ [1, m]
```

→ use summation properties and reduce

---

## Final Result

```
[m*(2*m+1)*(2*m-1)] / 3
```

---

## Code

```cpp
int oc = (n+1)/2;
int m = oc;

cout << m*1LL*(2*m+1)*(2*m-1)/3 << endl;
```
